TODO Vert.x and Angular.js
=============================

Basic TODO application,  microservices with Vert.x and Angular.js

## Technology used

- AngularJS 2
- Vert.x 3
- Java 8
- Maven 3

## Building All Docker Images

```bash
./build
```

## Starting All Docker Images

Start an ad-hoc cluster
```bash
./restart-cluster
```
